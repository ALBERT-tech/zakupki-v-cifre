# Закупки в цифре

Сайт онлайн-школы **«Закупки в цифре»** — практические курсы для заказчиков и поставщиков в госзакупках: ИИ в госзакупках, участие по 44-ФЗ и 223-ФЗ, бухгалтерия.

Владелец — ИП Герасимова Р.Г. (технологический партнёр Сбербанка). Эксперт ИИ-направления — Альберт Гизатуллин.

## Структура

- `index.html` — главная: hero, каталог курсов, как учимся, эксперты, форма записи.
- `course-ai.html` — главный курс «ИИ в госзакупках» (10 занятий).
- `styles.css` — дизайн-система (графит + зелёный, Manrope).
- `script.js` — навигация, анимации, форма → backend.
- `chatbot.js` — чат-ассистент школы (стриминг с backend).
- `backend/` — FastAPI (форма → Telegram, ассистент → AgentPlatform). **Деплоится на VPS отдельно, в этот публичный репозиторий не входит.**

## Запуск локально

```bash
python -m http.server 8200
# http://localhost:8200
```
